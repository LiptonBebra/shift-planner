# Ручной чек-лист тестирования Shift Planner

## Тест 1: Вход в систему
-  Открыть http://localhost:8080
-  Выбрать "Администратор" → появилось поле для пароля
-  Ввести пароль "admin123" → вход выполнен, показаны все смены
-  Нажать кнопку выхода → возврат на страницу входа
-  Выбрать "Иван Петров" (сотрудник) → поле пароля НЕ появляется
-  Нажать "Войти" → вход выполнен, показаны ТОЛЬКО смены Ивана
-  Нажать кнопку выхода → возврат на страницу входа

## Тест 2: Создание смены (админ)
-  Войти как админ
-  Выбрать сотрудника (например, Иван Петров)
-  Выбрать дату (например, завтрашний день)
-  Указать время начала (09:00) и конца (18:00)
-  Нажать "Создать" → смена появилась в списке
-  Попробовать создать смену с end_time < start_time (09:00 → 08:00) → ошибка "Время окончания должно быть позже"
-  Попробовать создать смену в то же время что и существующая → ошибка "Обнаружено пересечение"

## Тест 3: Редактирование смены (админ)
-  Нажать ️ Кнопку редактирования(карандаш) у любой смены
-  Проверить, что форма заполнилась данными этой смены
-  Изменить дату или время
-  Кнопка "Создать" изменилась на "Обновить смену"
-  Нажать "Обновить смену" → данные изменились в таблице

## Тест 4: Изменение статуса (админ)
-  Найти смену со статусом "Запланирована" (желтый фон)
-  Выбрать в выпадающем списке "Подтверждена" → статус изменился на зеленый
-  Выбрать "Отменена" → статус изменился на красный
-  Выбрать обратно "Запланирована" → статус вернулся на желтый

## Тест 5: Удаление смены (админ)
-  Нажать (корзину) у любой смены
-  Появилось окно подтверждения "Удалить смену?"
-  Нажать "OK" → смена исчезла из списка
-  Создать новую смену и нажать "Отмена" → смена осталась

## Тест 6: Фильтры (админ)
-  В фильтре по дате выбрать конкретную дату
-  Нажать "Применить" → показаны смены только за эту дату
-  В фильтре по сотруднику выбрать "Иван Петров"
-  Нажать "Применить" → показаны смены только Ивана
-  Выбрать и дату, и сотрудника → показаны смены по обоим условиям
-  Нажать "Сбросить" → показаны все смены

## Тест 7: Недельный просмотр (админ)
- Нажать кнопку "Неделя"
- Показаны смены только на текущую неделю (с понедельника по воскресенье)
- В счетчике смен отображается "(текущая неделя)"
- Нажать "Сбросить" → вернулись все смены

## Тест 8: Пагинация (админ)
- Убедиться что в базе больше 5 смен (если нет - создать еще)
- Внизу таблицы появились кнопки "← Назад" и "Вперед →"
- Показана информация "Страница 1 из X"
- Нажать "Вперед →" → показаны следующие 5 смен, страница стала 2 из X
- Нажать "Назад ←" → вернулись на первую страницу
- На первой странице кнопка "Назад" неактивна (серая)
- На последней странице кнопка "Вперед" неактивна (серая)
- Применить фильтр → пагинация обновилась под количество отфильтрованных смен
