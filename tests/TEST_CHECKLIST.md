# Ручной чек-лист тестирования Shift Planner

##  Тест 1: Вход в систему
-  Открыть http://localhost:8080
-   Выбрать "Администратор"
-   Ввести пароль "admin123" → вход выполнен
-   Выбрать "Иван Петров" (без пароля) → вход выполнен
-   Нажать кнопку выхода → возврат на страницу входа

## Тест 2: Создание смены (админ)
-   Войти как админ
-   Выбрать сотрудника, дату, время начала и конца
-   Нажать "Создать" → смена появилась в списке
-   Попробовать создать смену с end_time < start_time → ошибка
-   Попробовать создать пересекающуюся смену → ошибка

##  Тест 3: Редактирование смены (админ)
-   Нажать карандаш редактирования у любой смены
-   Изменить данные
-   Нажать "Обновить" → данные изменились

##  Тест 4: Изменение статуса (админ)
-   Выбрать статус "Подтверждена" → статус изменился
-   Выбрать статус "Отменена" → статус изменился

##  Тест 5: Удаление смены (админ)
-   Нажать на корзину
-   Подтвердить удаление → смена исчезла

##  Тест 6: Фильтры (админ)
-   Выбрать дату в фильтре → показаны смены только за эту дату
-   Выбрать сотрудника → показаны смены только этого сотрудника
-   Нажать "Сбросить" → все смены

##  Тест 7: Недельный просмотр (админ)
-   Нажать "Неделя" → показаны смены на текущую неделю

##  Тест 8: Пагинация (админ)
-   Если смен >5, появились кнопки "Назад"/"Вперед"
-   Нажать "Вперед" → следующая страница

##  Тест 9: Просмотр сотрудника
-   Выйти и зайти как сотрудник
-   Проверить что видны ТОЛЬКО свои смены
-   Проверить что нет кнопок создания/редактирования/удаления
