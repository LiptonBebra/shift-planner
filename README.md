# Shift Planner

Веб-приложение для планирования рабочих смен сотрудников.

## Функциональность

- Администратор: создание, редактирование, удаление смен
- Сотрудник: просмотр своих смен
- Фильтрация по дате, неделе и сотруднику
- Просмотр смен на текущую неделю
- Пагинация (по 5 смен на странице)
- Изменение статусов смен: Запланирована / Подтверждена / Отменена
- Адаптивный дизайн
- Вход на страницу администратора по паролю "admin123"

## Технологии

- Go 1.21
- PostgreSQL 15
- Gorilla Mux
- Tailwind CSS
- Docker / Docker Compose
